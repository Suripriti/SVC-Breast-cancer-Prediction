# Breast Cancer Prediction using SVC

This project predicts whether breast cancer is
Malignant or Benign using Support Vector Classifier (SVC).

---

# Dataset

Breast Cancer Wisconsin Dataset

https://www.kaggle.com/datasets/uciml/breast-cancer-wisconsin-data

Download and place:
data.csv

inside the project folder.

---

# Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Support Vector Classifier (SVC)

---

# Machine Learning Workflow

1. Load Dataset
2. Data Cleaning
3. Encode Labels
4. Feature Scaling
5. Train-Test Split
6. Train SVC Model
7. Evaluate Model
8. Predict Results
